## Prepare environment

- "node": ">=18"
- "pnpm": ">=8"
- Azure speech Service key && (OpenAi key || AlibabaTongyi key)

## Start up

1. cp env-sample.txt .env

2. Configure .env file

3. pnpm install

4. pnpm dev
